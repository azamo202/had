# منصة هدية للتخطيط الاستراتيجي والتشغيلي
### Hādiyah Strategic & Operational Planning Platform — v1.0 (Interactive Prototype)

منصة تفاعلية كاملة لإدارة ومتابعة واعتماد الخطة الاستراتيجية والتشغيلية لجمعية هدية (خدمة ضيوف الرحمن)، مبنية بـ **Vite + React**، بواجهة عربية بالكامل واتجاه من اليمين لليسار (RTL)، وبمستوى مؤسسي.

---

## التشغيل | Run

```bash
npm install
npm run dev
```

ثم افتح الرابط الذي يظهر في الطرفية (عادة `http://localhost:5173`).
Then open the URL printed in the terminal (usually `http://localhost:5173`).

للبناء للإنتاج | Production build:
```bash
npm run build
npm run preview
```

**المتطلبات | Requirements:** Node.js 18+.

## البيانات | Data model

**مصدر البنية هو ملف الخطة (Excel).** تم استخراج البنية فقط منه:
الأهداف الاستراتيجية ← الأهداف الفرعية ← المبادرات ← المشاريع التشغيلية ← مؤشرات الأداء (الأساس، المستهدف، التكلفة، المالك، الإدارة).

> **مهم:** الأعمدة التي تلي عمود "المستهدف" في الملف (التوزيع الشهري: يناير، فبراير…) **مُستبعدة تماماً من الاستيراد**. تُدار المتابعة الشهرية (المستهدفات الشهرية، الإنجاز الفعلي، التحديات، الشواهد) **بشكل ديناميكي داخل المنصة نفسها** عبر وحدة "المتابعة الشهرية".

آلية الاحتساب: **المُنجز الكلي = القيمة الأساسية (من الخطة) + مجموع الإنجاز الشهري المُدخل داخل المنصة**، ونسبة التحقق = المُنجز ÷ المستهدف، وتُشتق الحالة تلقائياً من الحدود القابلة للضبط في الإعدادات.

الأرقام الإجمالية: 8 أهداف · 28 هدفاً فرعياً · 37 مبادرة · 98 مشروعاً · 127 مؤشراً · 16 إدارة.

البيانات محفوظة محلياً في المتصفح (`localStorage`)؛ يمكن إعادة تعيينها من **الإعدادات ← إعادة تعيين البيانات**.

## الوحدات | Modules

- **لوحة المعلومات** — نظرة تنفيذية: تقدم استراتيجي/تشغيلي، حالات المشاريع، أداء الإدارات، المؤشرات المتأخرة، التنبيهات.
- **الخطة الاستراتيجية / التشغيلية** — عرض هرمي وجداول قابلة للتصفية والفرز.
- **الأهداف / المبادرات / المشاريع / المؤشرات** — صفحات قوائم وتفاصيل مترابطة.
- **المتابعة الشهرية** — النموذج التفاعلي الأساسي: إدخال الإنجاز الشهري، احتساب النسبة والحالة تلقائياً، تسجيل التحديات وإرفاق الشواهد، ثم حفظ كمسودة أو إرسال للمراجعة.
- **التحديات / الشواهد** — إدارة كاملة مع أنواع الشواهد ومسار الاعتماد (اعتماد/إرجاع).
- **التقارير** — 9 أنواع تقارير مع طباعة وتصدير Excel (CSV) وتصدير PDF (طباعة).
- **المساعد الذكي** — واجهة محادثة تحليلية على بيانات الخطة (نموذج قائم على القواعد).
- **الإشعارات / إدارة المستخدمين / الإعدادات** — مركز تنبيهات، إدارة حسابات وصلاحيات، وضبط قواعد الحالة وأنواع الشواهد والمظهر.
